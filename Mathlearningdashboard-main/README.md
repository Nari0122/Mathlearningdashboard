  # Math Learning Dashboard UI (Vite)

  This is a code bundle for Math Learning Dashboard UI. The original project is available at https://www.figma.com/design/4FRiOvibqrHrucMpvrmmV4/Math-Learning-Dashboard-UI.

  ## ⚠ 통합 안내

  **실제 운영 앱은 상위 폴더의 `math-dashboard-next`(Next.js) 단일 프로젝트로 통합되었습니다.**  
  로그인·학생/관리자 플로우·Mock 폴백은 Next 프로젝트에서 동일하게 동작합니다.  
  이 Vite 프로젝트는 참고/레거시용이며, 새 기능은 `math-dashboard-next`에서 진행해 주세요.

  ## Running the code (참고용)

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
